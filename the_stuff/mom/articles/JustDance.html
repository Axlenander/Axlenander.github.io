Getting into the game

So, now we can start getting started with the instructions

  1. Grab The Wii U Gamepad, The big one with a screen
  2. Touch the icon that looks like you!
  3. If there is an icon in the upper left corner other than Just dance, you need to follow these steps
    1. If The icon isn't just a blank square, then there is a game already in there and you need to remove it
    2. You need to walk up to the Wii U and press the Eject button
    3. Take the game and look at the disk, try to find the disk holder somewhere, if you can't find it, just place it somewhere
    4. Now, what we need to do now is find the Disk to play the game, it's going to be in the Cabinet under the T.V.
